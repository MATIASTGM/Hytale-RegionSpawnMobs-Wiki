# RegionSpawnMobs

RegionSpawnMobs is a Hytale server mod that gives admins full control over mob spawning through custom regions.

Instead of relying on global or player-centered spawning, this mod lets you define exact 3D areas where specific mobs can appear. Each region can have its own mob list, spawn weights, capacity, spawn mode, time conditions, weather conditions, terrain conditions and fixed SpawnPoints.

## What it does

RegionSpawnMobs allows you to create controlled spawn areas for:

- dungeons
- arenas
- forests
- caves
- farms
- boss rooms
- event areas
- natural habitats
- RPG survival worlds

Players do not directly receive mobs around them. Instead, players activate nearby regions, and the region handles spawning internally based on its configuration.

## Main Features

- Custom 3D spawn regions
- Player proximity activation
- Chunk-based mob capacity
- Per-region mob density
- Spawn weights per mob
- Spawn modes: Continuous, Wave and Threshold
- SpawnPoints for fixed spawn locations
- Spawn Markers for independent point-based spawns
- Zones and Cuts to include or exclude areas
- Parent region system for reusable configurations
- Time of day conditions
- Weather conditions
- Terrain type conditions
- Smart mob containment
- Region and chunk visualization tools
- In-game configuration menus
- JSON-based configuration
- Automatic backup system

## Project Status

RegionSpawnMobs is actively maintained and has moved beyond its beta stage as of version 1.1.8.

The core systems are stable and usable, but the mod is still under active development. New features, improvements and compatibility adjustments may continue to be added over time.
